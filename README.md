# Atelier — Privacy Policy

This repository hosts the Privacy Policy for the [Atelier](https://github.com/abdialimovtemur) Chrome Extension.

**Live page:** https://abdialimovtemur.github.io/atelier-privacy/

## Summary

Atelier does not collect any personal data. All user data (tasks, bookmarks, settings) is stored locally on the device using Chrome's built-in storage API.
